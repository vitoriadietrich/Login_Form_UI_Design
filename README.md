# Neumorphism Login Form UI Design using HTML & CSS | CodingNepal

Um projeto simples de **formulário de login com efeito neumorphism** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **login form moderno com estilo neumorphism**, utilizando sombras suaves para um visual elegante.

## 🛠️ Tecnologias

- HTML
- CSS (box-shadow, neumorphism)

## 🎯 Funcionalidades

- Formulário de login estilizado
- Efeito neumorphism
- Design moderno e minimalista

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
